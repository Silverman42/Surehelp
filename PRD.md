### Artisan Find Work Page (Dashboard)

- This pages has two top sections; Notification and Main Container
- Below are the features for the Notification section / component
  - Max width - 1448px
  - position - centralized
  - Border radius - 16px
  - It has a left aligned and right aligned content
  - The Left aligned content contains a fully rounded container which I call alert icon container (40px \* 40px, background - white) with a centralized icon (alert icon) (20px)
  - After the container is a text (color - #161616, size - 16px)
  - The Left aligned content contains a fully rounded container which I call notifcation close button (32px \* 32px, background - white) with a centralized icon (right-arrow-2.svg, 11px, color - #2275E6)
  - The Notification component has two states, warning and pending
  - In warning state
    - notification background - #FFF2F0
    - alert icon color - #FB583A
    - alert icon type - warning-2.svg
    - notification border color - transparent
  - In pending state
  - notification background - ##F9F9FB
  - alert icon color - #2275E6
  - alert icon type - warning-2.svg
  - notification border color - #F3F4F7
